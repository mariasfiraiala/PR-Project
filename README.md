# Jurnalized Multi Sensor Weather Station

## Run

```console
$ docker compose up --build
```

## Grafana

Grafana runs on `localhost:3000` with the following credentials:

User: admin \\
Password: admin

## API

The API runs on `localhost:55000`.

## Cleanup

```console
$ docker compose down --volumes
```
